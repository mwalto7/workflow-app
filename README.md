# Workflow

## Getting Started

### Dependencies

1. Nodejs
2. Postgresql

### Installation

1. Run the following commands to get started:

* ```sh
  $ git clone https://github.com/mwalto7/workflow-app.git
  $ cd workflow-app
  ```

2. Start the frontend:

* ```sh
  # Using yarn
  $ yarn && yarn start
  ```
* ```sh
  # Using npm
  $ npm install && npm start
  ```

3. Open a seprate terminal and start the server:

* ```sh
  # Using yarn
  $ yarn run server
  ```
* ```sh
  # Using npm
  $ npm run server
  ```
